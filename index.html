Instructions

  1. Press the START button. Simon will give you the first signal. Repeat the signal by pressing he same color lens.
  2. Simon will duplicate the signal and add one. Repeat these two signal by pressing he same color lenses, in order.
  3. Simon will duplicate these first two signals and add one.
  4. Continue playing as long as you can repeat each sequence of signals correctly.
  5. If you fail to repeat a sequence exactly, you lose. Click the RESET button and then the START button to play again.
